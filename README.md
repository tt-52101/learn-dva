##项目介绍

该项目仿照ant-design官网制作，使用dva脚手架开发
主要想体验下react配合antd的开发流程
##特别说明
ant-design官网是基于github上的.md文件完成，将markdown格式的文件转为网页展现，使用的工具是bisheng
##具体内容
顶部导航，侧边栏，主体内容，底部
侧边栏上部分是模仿的组件介绍，挑了几个
侧边栏下部分是自己写的其他router内容，配合其他插件的使用
